# Documentation technique — Node.js Week 3

Semaine 3 : **base de données** et structure avancée. Ce dossier contient les **post-lesson activities** (base) et le **week-projet TodoTrackerApi**.

## 1. Post-lesson activities — base

Squelette avec **config DB**, **models**, **routes**, **services**, **data**, **middlewares** : `src/config`, `src/models`, `src/controllers`, `src/routes`, `src/services`, `src/data`, `src/middlewares`.

### Lancement

- Dans `post-lesson activities/base` : `npm install`, configurer la DB, puis `npm start` ou `npm run dev`.

## 2. Week-projet — TodoTrackerApi

Projet complet TodoTracker : serveur avec **config**, **controller**, **model**, **routes**, **service**, **middlewares**, **public**.

### Stack

- **Express**, base de données (config dans `config/`, modèles dans `model/` ou `models/`).
- Architecture : routes → controllers → services → models.

### Lancement

- Dans `week-projet/TodoTrackerApi/serveur` : `npm install`, configurer la DB, puis `npm start` ou `npm run dev`.
